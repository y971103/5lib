[ 프로젝트 - 전자책 도서관. e-book 뷰어 및 각종 기능, 정보 제공 사이트  ]

<* 개발환경>
- Java 1.8 
- Spring 4.3.6
- Mybatis 3.4.6
- jQuery 3.3.1
- SQL - oracle developer
- HTML, CSS, JS - VS code

- "team5" project with GitHub tool.

<* 주요 기능>
- 메인, 도서관 (전체 서재), 독자 게시판, 내 서재 (마이 페이지)

# 메인
- 네이게이션 바 구현 (전체적인 화면은 반응형 디자인으로 세팅. (삼단 바 등 구현) - Martin 탬플릿 활용
- 회원가입, 로그인, 로그아웃	(한승 업로드 파일 활용 - 로그인, 회원가입 동시 구현 탬플릿)
- 눈 보호 모드 기능 (다크모드)

# 도서관 (전체 서재)
- 추천 도서, 베스트 셀러 노출 기능 (조회수로 랭크한 베스트 셀러)
- 선택한 도서 상세 페이지 - 내 서재에 담기 기능, 한 줄 리뷰, 좋아요 싫어요 기능, 추천 도서 (마우스 커서 이동 시 움직임 기능)
	- 추천 기능은 Ajax 이용 

# 독자 게시판 
- 글쓰기형 게시판 모드 / 썸네일형 게시판 모드 (선택 가능) - 코드 Custom
- 게시판 글쓰기, 수정, 삭제
- 로그인 시 등록 가능 (확인은 인터셉터 이용)

# 내 서재 (마이 페이지)	- HTML5UP 활용
- 내 회원 정보
    - 가입시 선택한 프로필 이미지 출력기능
    - 회원 정보 수정 기능
    
- 리뷰 / 북마크(책장) / 통계 카테고리로 나눠서 구성.
	- 전자 책 뷰어 (읽기 모드) - 리디움 클라우드 리더*
	- 밑줄, 그리기 기능, 글꼴 변경 기능
	- 내 독서습관 모드 ex. 독서 시간 추이 그래프 (통계)
	- 공유 게시 기능 (페이스북, 트위터 등)

* OPEN API
- 리디움 클라우드 리더 (Epub)
